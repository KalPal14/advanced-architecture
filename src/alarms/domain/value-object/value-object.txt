Value-object - концепт родом из Domain-Driven Design (DDD). 
Это неизменяемый тип, который определяется своими свойствами,
а не уникальной идентичностью. 
В отличие от сущностей, которые различаются по идентификаторам, 
объекты-значения различаются по атрибутам. 
Два объекта значения считаются равными, если все их атрибуты одинаковы.
